Wildflowers API Documentation

State

if you want to find all wildflowers by state use localhost:3000/api/stateName and fill in stateName for that state

ex. Nevada wildflowers: localhost:3000/api/Nevada (not case sensitive)

List of States Available:

  • California
  • Nevada
  • Arizona
  • New Mexico
  • Colorado
